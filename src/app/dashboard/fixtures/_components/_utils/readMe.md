# Folder Overview

Utility functions and components shared across fixture components.

## Files

- `types.ts`: Common TypeScript types for sorting and filtering
- `useSorting.ts`: Custom hook for managing table sorting state and logic
- `SearchInput.tsx`: Reusable search input component with clear button
- `dateUtils.ts`: Date formatting utilities
- `statusUtils.tsx`: Status badge and row color utilities for fixtures
- `fixtureUtils.ts`: Helper functions for fixture data manipulation

## Relations

- Parent folder: [../readMe.md](../readMe.md)
- Used by: `AssociationSelector`, `AssociationFixturesTable`, and other fixture components

## Dependencies

- Internal: React hooks, UI components
- External: `date-fns`, `lucide-react`
